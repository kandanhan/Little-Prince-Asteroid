# 별솜이의 작은 별 — 브랜드 바이블 (BRAND)

> 어린왕자 IP를 피해 **독자 브랜드화**한 정체성 문서. 게임플레이(작은 행성 가꾸기·교감·창작·힐링)는 유지하고, 생텍쥐페리 고유 표현만 오리지널로 대체.
> **마스코트 주도 브랜드** · 잠정 확정명 **별솜이(Sombyeol)** *(또롱이/포실이로 교체 가능. 상업 출시 전 KIPRIS·Play·도메인 최종 확인은 🙋 당신 단계)*

---

## 1. 한 줄 정의
- **이름(KO)**: 별솜이의 작은 별 · **이름(EN)**: Sombyeol — A Tiny Star
- **태그라인(KO)**: “솜구름 친구와 가꾸는 나만의 작은 별”
- **태그라인(EN)**: “Tend your tiny star with a little cloud friend.”
- **포지셔닝**: 경쟁이 붐비는 코지 정원 게임 속에서 **‘구형 행성 산책 + 솜구름 마스코트 교감 + 그림/음악 창작’** 조합으로 차별화. 아이콘=별솜이 얼굴(인지도).

## 2. 마스코트 — 별솜이 (Sombyeol)
- **형태**: 둥근 **솜구름** 몸 + 작은 둥근 귀 2개(여우도 토끼도 아닌 모호한 독자형) + 점 눈 + 발그레 볼. 팔다리 거의 없이 **둥실** 떠다님. **꼬리 끝이 작은 별** 모양(별+솜 연결).
- **성격**: 말 없이 곁을 지키는 다정한 동반자. 쓰다듬으면 눈을 ^^ 로 접고 하트를 낸다.
- **팔레트**: 크림화이트 솜(#FBF6EC) + 파스텔 민트(#BFE3D0)/피치(#F6C9B8) 포인트 + 별꼬리 노랑(#FFD66B).
- **애니메이션**: 둥실둥실 상하 bob(웹 prince의 bob 로직 재사용), 따라다니기, 쓰다듬기 반응, 잠들기.
- **반드시 피할 것(IP 안전)**:
  - ❌ 사람/소년 형태·금발·코트·목도리 (어린왕자 회피)
  - ❌ 파란 몸+비행모/안경 (뽀로로 회피 — '포롱이' 폐기 이유)
  - ❌ 기존 마스코트 클리셰(곰/판다/노란 새 등) 답습

## 3. 플레이어 & 세계관 (오리지널)
- **플레이어 = ‘별지기’**: 작고 둥근 실루엣(니트 모자·등불). *왕자 디자인 완전 폐기.* (원하면 카메라만 두고 캐릭터를 별솜이로 대체하는 변형도 가능)
- **무대**: 손바닥만 한 **작은 별** 하나를 거닐며 가꾼다. (구형 중력 보행=기존 코드 유지)
- **행성 명명(‘B-612’ 대체)**: 고유 번호 대신 **시적 이름** — 첫별 · 노을별 · 잠든별 · 들꽃별 · 눈별 · 밤별 … (테마 6종과 1:1로 연결)
- **핵심 정서**: 처벌·경쟁 없는 **느린 돌봄**. 며칠 비워도 “돌아와서 반가워요”.

## 5. 오리지널 카피 — 책 인용 대체 (그대로 코드에 투입 가능)
> 기존 `items.ts/PlanetItemCatalog`의 힌트에 어린왕자 인용(“단 하나뿐인 나의 장미”, “노을을 마흔네 번”, “길들인다는 건…”, “웃을 줄 아는 별”, “사막을 아름답게 하는 건 숨은 우물” 등)이 섞여 있어 **전면 교체**.

| 아이템 | 기존(인용·위험) | 오리지널 대체 |
|---|---|---|
| 장미 | 세상에 단 하나뿐인 나의 장미 | 오늘도 한 송이, 별이 환해졌어요 |
| 새싹 | 작을 때 돌봐주면 친구가 돼요 | 작은 떡잎, 내일이 기대돼요 |
| 나무 | 그늘을 만들어 주는 친구 | 그늘 아래 잠깐 쉬어가요 |
| 가로등 | 밤이 오면 스스로 불을 밝혀요 | 밤이 오면 깜빡, 스스로 불을 켜요 |
| 벤치 | 노을을 마흔네 번 바라보던 자리 | 나란히 앉아 노을을 봐요 |
| 여우 | 길들인다는 건 관계를 맺는 것 | 천천히 다가가면, 어느새 친구 |
| 양 | 상자 속에 잠든 나의 양 | 포근한 털뭉치가 데구르르 |
| 별 | 웃을 줄 아는 별 하나 | 콕 놓으면 반짝, 웃는 별 |
| 수정 | 빛을 머금은 작은 결정 | 빛을 머금은 작은 결정 *(일반 표현, 유지 OK)* |
| 버섯 | 비 온 뒤 돋아난 작은 우산 | 비 온 뒤 쏙 돋은 우산 |
| 들꽃 | 바람에 흔들리는 들꽃 무리 | 바람에 살랑이는 들꽃 무리 |
| 우물 | 사막을 아름답게 하는 건 어딘가 숨은 우물 | 별 어딘가 숨어 있는 시원한 우물 |
| 화산 | 매일 청소하면 따뜻한 아침밥을 지어줘요 | 아침마다 보글보글, 따뜻한 김 |

**인트로 문구(‘가장 중요한 건…’ 인용 대체)**: “작은 별 하나, 천천히 가꿔봐요.”

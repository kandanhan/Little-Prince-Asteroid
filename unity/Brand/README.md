# Orblet · 별마실 — 브랜드 에셋

자동 생성 1차 에셋(SVG). 스토어급 최종본은 디자인 툴(Adobe Express/Canva)에서 OFL 폰트로 다듬어 PNG 내보내기 권장.

## 파일
- `app-icon.svg` — 앱 아이콘 마스터(1024). 크림 배경 + 행성/오비트 + 마스코트 **별솜이**.
- `wordmark.svg` — 워드마크 레이아웃 목업(‘O’=작은 행성+별솜이). **텍스트는 시스템 폰트 폴백** → 최종은 아웃라인 폰트로 교체.

## 팔레트
| 이름 | HEX |
|---|---|
| 크림(배경) | `#FBF6EC` |
| 피치 | `#F6C9B8` |
| 민트 | `#BFE3D0` |
| 별노랑 | `#FFD66B` |
| 행성 그린 | `#8FCBA3` |
| 글자/밤보라 | `#4A4660` |

## 폰트 (전부 OFL, 상업 임베드 가능)
- 워드마크(EN): 둥근 지오메트릭 산세(예: **Baloo 2 / Quicksand**) — 라이선스 확인 후 아웃라인화
- 헤더(KO): **Gowun Dodum/Batang**
- 본문(KO): **Pretendard** · 폴백 **Noto Sans KR**

## 내보내기 규격 (ROADMAP P8)
- Play 앱 아이콘 512×512 PNG · 적응형 아이콘(전경/배경 분리, 안전영역 66%)
- 피처 그래픽 1024×500 · 폰 스크린샷(긴변≤짧은변×2)
- iOS 1024×1024(추후)

> 마스코트 정밀 일러스트(고해상 표정/포즈)는 디자인 툴 또는 AI 이미지로 보강. 본 SVG는 컨셉·색·구도 기준선.
